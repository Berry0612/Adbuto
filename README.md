# Adbuto 

Adbuto 是一個輕量級的生產力工具，透過 USB 連線將 Android 手機的通知即時同步到 Windows 電腦，並支援自動開啟 USB 網路分享功能。

---

##  下載與安裝 (Installation)

### 第一步：電腦端 (Windows)

* **系統需求**：Windows 10 / 11
* **必要元件**：請確保已安裝 [**.NET 8 Desktop Runtime**](https://dotnet.microsoft.com/en-us/download/dotnet/8.0) (請選擇 `Run desktop apps` 的 **x64** 版本)。
* **執行方式**：執行 `PCLinkNative／Program.cs`。

> [!IMPORTANT]
> **注意**：程式啟動後會在背景執行，請查看電腦右下角 **系統列 (System Tray)** 的小圖示進行操作。

---

### 第二步：手機端 (Android)

1.  **下載 App**：請點擊 [**這裡下載 APK 檔案**](你的下載連結) 並安裝至手機。
2.  **開啟權限**：啟動 App，點擊「權限檢查」按鈕，並勾選允許 **「通知存取權限」**。
3.  **開發人員設定 (關鍵步驟)**：
    * 進入手機的 `設定` -> `開發人員選項`。
    * 開啟 `USB 偵錯 (USB Debugging)`。

---

##  如何使用 (Usage)

請依照下列步驟確保連線成功：

| 步驟 | 操作說明 | 狀態檢查 |
| :--- | :--- | :--- |
| **1** | 啟動電腦端程式 | 確認右下角系統列出現圖示 |
| **2** | 使用 USB 線連接手機與電腦 | 手機彈出偵錯授權時請點選「允許」 |
| **3** | 確認連線狀態 | 手機 App 應顯示 `Connected` 或呈現 **綠燈** |
| **4** | 完成連線 | 電腦右下角會跳出 `連線成功` 通知 |

###  （選用）功能
* **網路分享**：在電腦右下角圖示上按 **右鍵**，可勾選 `Enable USB Tethering` 來自動控制網路分享。
