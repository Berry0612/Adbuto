📱 PC-Link Native (Android USB 通知同步工具)
這是一個輕量級的生產力工具，透過 USB 連線將 Android 手機的通知同步到 Windows 電腦，並支援自動開啟 USB 網路分享。

📥 下載與安裝 (Installation)
第一步：電腦端 (Windows)
系統需求：Windows 10 或 Windows 11。

必要元件：請確保您的電腦已安裝 .NET 8 Desktop Runtime (選擇 Run desktop apps 的 x64 版本)。

下載程式：前往 Releases 下載 PC-Link-Client.zip。

執行：解壓縮後，點擊 PCLinkNative.exe。

注意：程式會在背景執行，請查看右下角系統列的小圖示。

第二步：手機端 (Android)
下載 App：前往 Releases 下載 PC-Link-Android.apk 並安裝到手機。

開啟權限：

開啟 App，點擊權限檢查按鈕，允許「通知存取權限」。

開發人員設定 (重要)：

開啟手機的「設定」->「開發人員選項」。

開啟 「USB 偵錯 (USB Debugging)」。

🚀 如何使用 (Usage)
確認電腦端程式已啟動（右下角有圖示）。

使用 USB 傳輸線將手機連接至電腦。

手機 App 應會顯示 "Connected" 或綠燈狀態。

電腦右下角會跳出連線成功通知。

(選用) 在電腦右下角圖示按右鍵，可勾選 "Enable USB Tethering" 來控制網路分享。
